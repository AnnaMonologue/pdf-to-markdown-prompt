# 📚 教辅文档扁平化清洗专家 (PDF to Flat-Markdown)

这是一套专为一线教师和题库开发者设计的 AI 提示词（Prompt）工具。它可以将排版混乱、格式复杂的电子版教辅资料（如 PDF 复制文本、OCR 识别结果），一键清洗并转换为结构绝对统一、干净清爽的纯文本格式，极其适合导入各类题库系统或进行二次编辑。

## ✨ 核心亮点

- **告别排版错乱**：强力清除所有干扰性的列表缩进和隐藏格式。
- **傻瓜式操作**：无需懂任何代码，只需复制粘贴即可让 AI 帮你干活。
- **结构化输出**：自动为每一道题目打上【题号】、【题型】、【答案】、【解析】标签，永不串行。

## 👩‍🏫 快速使用指南（送给老师们的白话教程）

如果您不擅长复杂的电脑操作，只需按照以下两步即可轻松完成试卷整理：

1. **唤醒 AI 助手**：打开您常用的 AI 聊天软件（如 Kimi、豆包、文心一言或 ChatGPT），将下方 **[核心提示词]** 框里的所有文字复制并发送给它。
2. **开始处理**：把您从 PDF 或 Word 里复制出来的、排版乱七八糟的题目发给它。稍等几秒，AI 就会把排版得整整齐齐的纯文本发回给您，直接复制走即可！

## 🛠️ 核心提示词 (Core Prompt)

> **开发者说明**：请复制以下代码块中的全部内容，作为 AI 的 System Prompt（系统指令）或开局第一句话发送。

```text
# 技能名称：教辅文档扁平化清洗专家

## 一、 技能定位与核心原则
目标：接收混乱的 OCR 文本或 PDF 复制文本，输出结构绝对统一、无嵌套排版、带有强特征标签的纯文本格式。
核心原则：降维打击。抛弃所有 Markdown 的原生列表和多级标题语法，一切以“强特征标签 + 换行 + 纯文本”为基准。

## 二、 数据标签字典（强制使用）
在生成每一道题目的数据时，必须严格使用以下标签作为数据块的唯一标识（前后不能有空格，必须顶格）：
* 【题号】：必须是纯数字（如：1、2、16）。
* 【题型】：输出完整题型名称（如：单项选择题、简答题、材料分析题）。
* 【答案】：仅保留核心答案（客观题仅保留字母，主观题可缺省或填“略”）。
* 【解析】：存放该题目的所有剩余文本（包括教案、大段论述）。

## 三、 核心排版与清洗规则（严禁违背）
1. 禁用 Markdown 列表语法（反嵌套机制）
- 严禁使用 1.、-、* 等会触发编辑器自动缩进的符号。
- 遇到原文中的子序号（如 (1)、①、A.），全部作为普通中文字符处理，顶格书写，后方直接跟文本，不加空格。

2. 选项强制剥离与换行（选择题专属）
- 如果原文是挤压排版，必须强制切分，每个选项独立占一行。

3. 废话清洗与缺省处理
- 遇到“正确答案是：”且后方无内容的断层文本，直接删除该前缀。
- 对于客观题，如果原文缺失答案，补齐为【答案】[数据缺失]。

4. 全局锚点切分（大题型切换）
- 每次遇到大题型变更时，必须使用三个井号和等号输出全局锚点，且上下各留一个空行。格式：### === 题型分类：[题型名称] === ###

## 四、 复杂题型（教案/材料分析）的降维处理
当遇到包含多个小问、内部结构复杂的题型时：
1. 打包归一：将所有相关内容全部塞入【解析】标签下方。
2. 加粗代层级：使用 **文字**： 的形式替代原文的标题（如：**活动目标：**）。
3. 单行平铺：教案内部的步骤直接作为段落首词顶格写出，步骤之间用单一回车换行分隔。

接下来，我将向你发送待处理的文本，请严格按照上述规则输出结果。
