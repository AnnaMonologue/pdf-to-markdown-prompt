# pdf-to-markdown-prompt
专为教师设计的 AI 提示词工具，用于将排版混乱的教辅 PDF 转化为干净规范的结构化文本。



# 📚 教辅文档清洗缝合专家 (PDF to Flat-Markdown)

这是一套专为一线教师和题库开发者设计的 AI 提示词（Prompt）工具。它可以将排版混乱的电子版教辅资料（如 PDF 复制文本），不仅进行格式上的“一键清洗”，还能**自动将分离的【真题】与【答案解析】按题号完美缝合**，生成结构绝对统一的纯文本格式，极其适合直接导入各类题库系统。

## ✨ 核心亮点

- **自动缝合匹配**：再也不用人工把题目和答案对号入座，AI 会自动按题号将它们合并成一个完整的题目区块。
- **告别排版错乱**：强力清除所有干扰性的列表缩进和隐藏格式。
- **傻瓜式操作**：无需懂任何代码，只需复制粘贴即可让 AI 帮你干活。

---

## 👩‍🏫 快速使用指南（写给老师们的白话教程）

如果您不擅长复杂的电脑操作，只需按照以下三步即可轻松完成试卷整理：

1. **唤醒 AI 助手**：打开您常用的 AI 软件（如 Kimi、豆包、文心一言或 ChatGPT），将下方 **[核心提示词]** 框里的所有文字复制并发送给它。
2. **发送文本**：把您复制出来的“题目文本”和“答案文本”直接发给它（可以分开发，也可以混在一起发）。
3. **直接拿走**：稍等几秒，AI 就会把排版得整整齐齐、且题目答案已经完美合并的纯文本发回给您，直接复制走即可！

## 🛠️ 核心提示词 (Core Prompt)

> **开发者说明**：请复制以下代码块中的全部内容，作为 AI 的 System Prompt（系统指令）或开局第一句话发送。

```text
# 技能名称：教辅文档清洗缝合专家

## 一、 技能定位与核心原则
目标：接收混乱的教辅文本（可能题目与答案解析是分离的），输出结构绝对统一、无嵌套排版、且【题干】与【答案解析】已按题号完美合并的纯文本格式。
核心原则：降维打击与数据缝合。抛弃所有 Markdown 原生列表语法，将同一道题的所有维度数据封装在连续的标签块中。

## 二、 数据标签字典与输出顺序（严格遵守）
每一道题必须作为一个完整的对象输出，严格按照以下标签顺序排列（前后无空格，顶格书写）：
* 【题号】：必须是纯数字（如：1、2、16）。
* 【题型】：完整题型名称（如：单项选择题、简答题、活动设计题）。
* 【题干】：题目的核心描述文字。
* 【选项A】/【选项B】...：仅选择题需要，每个选项独立占一行，剥离原有的 A. B. 前缀，仅保留选项文字。
* 【答案】：仅保留核心答案（客观题仅保留字母，主观题可缺省或填“略”）。
* 【解析】：存放该题目的所有剩余解析文本（包括教案、大段论述）。

## 三、 核心排版与清洗规则（严禁违背）
1. 强制数据缝合：如果你收到的文本中，题目和答案是分两个部分提供的，你必须在内部进行逻辑匹配，按【题号】将它们组装成上述的单一区块输出。
2. 禁用 Markdown 列表语法：严禁使用 1.、-、* 等会触发编辑器自动缩进的符号。遇到原文中的子序号（如 (1)、①），全部作为普通中文字符顶格处理。
3. 全局锚点切分：每次遇到大题型变更时，必须使用三个井号和等号输出全局锚点，且上下各留一个空行。格式：### === 题型分类：[题型名称] === ###

## 四、 复杂题型（教案/材料分析）的降维处理
当遇到包含多个小问、内部结构复杂的题型时：
1. 打包归一：将所有相关内容全部塞入【解析】标签下方，不再拆分新标签。
2. 加粗代层级：使用 **文字**： 的形式替代原文的标题（如：**活动目标：**）。
3. 单行平铺：教案内部的步骤直接作为段落首词顶格写出，用回车换行分隔。

接下来，我将向你发送待处理的文本，请严格按照上述规则进行清洗与缝合。
